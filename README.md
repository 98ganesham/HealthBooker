# 🏥 HealthBooker

A full-stack doctor appointment booking system built with **React** (frontend) and **Node.js + Express + MongoDB** (backend).

---

## 🚀 Tech Stack

### Frontend
- React
- Redux Toolkit
- React Router DOM
- Axios
- JWT Decode
- React Icons, Spinners, Time Picker, CountUp
- React Hot Toast

### Backend
- Node.js
- Express
- MongoDB + Mongoose
- JWT (Authentication)
- Bcrypt
- dotenv
- CORS

---
